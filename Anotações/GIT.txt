Criar nova pasta
(vc seleciona a pasta q vc esta e cria um arquivo)

git init  (inicialiaza repositorio do zero)
git clone (copia repositorio)

git pull ( pucha alteracóes feitas por outros)

apos editar o arquivo e vc salvar o que vc quer vc adiiona para salvar

git add arquivo

depois salvar commit

git commit -m "coment?rio qualquer"

…or create a new repository on the command line

git remote add origin https://github.com/almirdapper/teste2.git
git push -u origin master
